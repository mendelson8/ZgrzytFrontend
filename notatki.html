1. Zmienne CSS i Kolorystyka (1.5 pkt)

Zarządzanie z jednego miejsca, kolory granatowy/jasnoszary, odwrócenie przy zaznaczeniu.
Aby zarządzać kolorami z jednego miejsca, definiujemy tzw. CSS Custom Properties (zmienne) w pseudo-klasie :root. Do stylowania zaznaczonego tekstu używamy selektora ::selection. 
/* Definiujemy zmienne na samej górze CSS */
:root {
    --main-color: #000080; /* Granatowy */
    --bg-color: #d3d3d3;   /* Jasnoszary */
}

body {
    color: var(--main-color);
    background-color: var(--bg-color);
}

/* Pseudo-element działający przy zaznaczaniu tekstu myszką */
::selection {
    background-color: var(--main-color);
    color: var(--bg-color);
}

2. Panel Logo (1.5 pkt)

Flexbox, łamanie wiersza, własna czcionka.
Hasło ZAWSZE podzielone na 2 wiersze oznacza użycie tagu <br>. Zmiana układu (obok siebie -> pod sobą) to klasyczny przypadek dla Flexboxa i tzw. Media Queries. Własną czcionkę ładujemy przez @font-face. 
/* Ładowanie czcionki z pliku */
@font-face {
    font-family: 'Caveat';
    src: url('caveat-regular.woff2') format('woff2'); /* dostosuj format */
}

.logo-panel {
    display: flex;
    align-items: center; /* Wyrównanie w pionie */
    justify-content: center;
    gap: 20px;
}

.motto {
    font-family: 'Caveat', cursive;
}

/* Responsywność: Gdy ekran jest węższy niż 768px, układamy pionowo */
@media (max-width: 768px) {
    .logo-panel {
        flex-direction: column;
        text-align: center;
    }
}

3. Panel Filtrów (2.5 pkt)

Select, disabled selected, optgroup, value kodowanie, reset form.
To pułapka na wiele punktów! 1. Aby przycisk "Wyczyść" działał naturalnie bez JS, wszystko musi być wewnątrz tagu <form>, a przycisk musi mieć type="reset". 2. Niewybieralny tytuł robimy flagami disabled selected hidden. 3. "Zakodowane wartości" oznaczają, że wpisujesz kod w atrybut value="1". 4. Kategorie robimy tagiem <optgroup label="...">. 
<form class="filters-panel">

    <select name="academic-title">
        <!-- Niewybieralny, domyślny tytuł -->
        <option value="" disabled selected hidden>Tytuł/stopień naukowy</option>

        <!-- Wartość dla maszyny (value="2"), tekst dla człowieka (Doktor) -->
        <option value="2">Doktor</option>
        <option value="3">Profesor</option>
    </select>

    <select name="topic">
        <option value="" disabled selected hidden>Obszar zainteresowań</option>

        <!-- Kategorie (niewybieralne) grupujące opcje -->
        <optgroup label="Matematyka">
            <option value="1">Obliczenia numeryczne</option>
        </optgroup>
    </select>

    <!-- Przycisk przywracający domyślne opcje formularza -->
    <button type="reset">Wyczyść</button>

</form>

4. Siatka i Kafelki (2.5 pkt)

Siatka 4 kolumn, przerywane zaokrąglone ramki, zdjęcia równej wysokości, padding/gap.
Użycie display: grid z opcją repeat(4, 1fr) automatycznie dzieli przestrzeń na 4 równe kolumny. Przezroczystość zdjęć i ich jednakową wysokość bez niszczenia proporcji (spłaszczania) załatwia object-fit: cover. 
/* --- SIATKA --- */
.promoters-grid {
    display: grid;
    grid-template-columns: repeat(4, 1fr); /* Dokładnie 4 kolumny */
    gap: 30px; /* Odpowiednie rozdzielenie (światło między kafelkami) */
}

/* --- KAFELEK --- */
.card {
    border: 2px dashed var(--main-color); /* Przerywana ramka */
    border-radius: 15px; /* Zaokrąglone rogi */
    padding: 20px; /* Światło wewnątrz kafelka */
}

/* --- ZDJĘCIA RÓWNEJ WYSOKOŚCI --- */
.card img {
    width: 100%;
    height: 250px; /* Sztywna wysokość dla wszystkich */
    object-fit: cover; /* Przycina, ale NIE ZNIEKSZTAŁCA proporcji! */
    border-radius: 8px;
}

5. Semantyka Tytułów i Skrótów (1.5 pkt)

Związanie zdjęcia z podpisem, rozwijany skrót po najechaniu.
To jest moment, w którym prowadzący sprawdza, czy znasz tagi HTML5. Zamiast pakować zdjęcie i nazwisko w puste <div>, należy użyć <figure> oraz <figcaption>. Do skrótów służy tag <abbr>. 
<!-- Poprawne semantycznie połączenie zdjęcia z osobą -->
<figure class="card">
    <img src="promotor1.jpg" alt="Zdjęcie dr. Jana Kowalskiego">

    <figcaption>
        <h3>dr Jan Kowalski</h3>
    </figcaption>

    <!-- Słowa kluczowe ze skrótami -->
    <div class="keywords">
        Zainteresowania:
        <!-- Tag abbr z atrybutem title robi automatyczny tooltip (podpowiedź) -->
        <abbr title="Artificial Intelligence">AI</abbr>,
        <abbr title="Machine Learning">ML</abbr>
    </div>
</figure>
