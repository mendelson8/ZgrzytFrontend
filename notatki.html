WPF: Contact Manager - Zrozumieć Kod

Zamiast wklejać gotowce, przeanalizujmy, dlaczego używamy konkretnych mechanizmów zdefiniowanych w wymaganiach zadania (PDF).

1. Ustawienia Okna i Podział (Layout)

Po co nam Grid i RowDefinitions?

Wymaganie z PDF mówi jasno: "Pasek menu na górze okna, reszta okna podzielona na zakładki". W WPF nie używamy pozycjonowania pikselowego. Używamy <Grid>, który działa jak niewidzialna tabela.

Plik: MainWindow.xaml (Fragment Głównego Układu)
<Window x:Class="ContactManager.MainWindow"
        Title="Contact Manager"
        Height="600" Width="800"
        MinWidth="500" MinHeight="500"
        WindowStartupLocation="CenterScreen">

    <Grid>
        <!-- Definicja naszej tabeli (2 wiersze) -->
        <Grid.RowDefinitions>
            <RowDefinition Height="Auto"/>
            <RowDefinition Height="*"/>
        </Grid.RowDefinitions>

        <!-- Wiersz 0: Menu -->
        <Menu Grid.Row="0">
            <MenuItem Header="File">
                <MenuItem Header="Exit" Click="MenuItem_Exit" />
            </MenuItem>
        </Menu>

        <!-- Wiersz 1: Zakładki zajmujące resztę ekranu -->
        <TabControl Grid.Row="1">
            <TabItem Header="List"> <ListBox/> </TabItem>
            <TabItem Header="Grid"> <DataGrid/> </TabItem>
        </TabControl>
    </Grid>
</Window>

2. Powiadomienia Widoku (INotifyPropertyChanged)

Dlaczego ObservableCollection nie wystarcza?

Samo użycie ObservableCollection<Contact> informuje widok tylko o tym, że element został dodany lub usunięty z listy. Nie powiadamia o edycji wartości (np. zmianie nazwiska). Używamy do tego interfejsu INotifyPropertyChanged.

Plik: Contact.cs
using System.ComponentModel;
using System.Runtime.CompilerServices;

public class Contact : INotifyPropertyChanged
{
    private string name;

    public string Name
    {
        get { return name; }
        set
        {
            name = value;
            OnPropertyChanged();
        }
    }

    public event PropertyChangedEventHandler PropertyChanged;
    protected void OnPropertyChanged([CallerMemberName] string propertyName = null)
    {
        PropertyChanged?.Invoke(this, new PropertyChangedEventArgs(propertyName));
    }
}

3. Magia: ComboBox ze zmiennymi typu Enum

Jak wyciągnąć "Male" i "Female" do listy rozwijanej?

Aby uniknąć brzydkiego kodu w C#, korzystamy z ObjectDataProvider, ładując nasz Enum bezpośrednio do Window.Resources i bindując jako ItemsSource.

Plik: AddContactWindow.xaml
<Window.Resources>
    <ObjectDataProvider x:Key="GenderList" MethodName="GetValues" ObjectType="{x:Type TypeName=System:Enum}">
        <ObjectDataProvider.MethodParameters>
            <x:Type TypeName="local:Gender"/>
        </ObjectDataProvider.MethodParameters>
    </ObjectDataProvider>
</Window.Resources>

<Grid>
    <ComboBox ItemsSource="{Binding Source={StaticResource GenderList}}"
              SelectedItem="{Binding Path=Gender}" />
</Grid>

4. Kolorowanie co drugiego wiersza (AlternationIndex)

Dlaczego używamy Style Triggers?

By wiersze automatycznie kolorowały się na zmianę (niezależnie od tego, jak sortujemy czy usuwamy dane ze środka), ustawiamy w ListBox atrybut AlternationCount="2". Resztę załatwia Trigger.

Plik: MainWindow.xaml
<ListBox ItemsSource="{Binding Contacts}" AlternationCount="2">
    <ListBox.ItemContainerStyle>
        <Style TargetType="ListBoxItem">
            <Style.Triggers>
                <Trigger Property="ItemsControl.AlternationIndex" Value="0">
                    <Setter Property="Background" Value="#FFAFC5FF" />
                </Trigger>
                <Trigger Property="ItemsControl.AlternationIndex" Value="1">
                    <Setter Property="Background" Value="#FF75A1FF" />
                </Trigger>
            </Style.Triggers>
        </Style>
    </ListBox.ItemContainerStyle>
</ListBox>

5. Zaawansowana Walidacja (IValidation)

Architektura rozszerzalna przez pliki DLL

Wykorzystujemy interfejs IValidation oraz dziedziczymy po ValidationRule, aby aplikacja pozwalała na wstrzykiwanie nowych zasad walidacji bez modyfikowania głównego kodu okna.

Plik: PhoneValidationRule.cs
public interface IValidation
{
    string Name { get; }
    string Description { get; }
}

public class PhoneValidationRule : ValidationRule, IValidation
{
    public string Name => "Format Telefonu";
    public string Description => "Format musi wynosić XXX-XXX-XXX.";

    public override ValidationResult Validate(object value, CultureInfo cultureInfo)
    {
        string phone = value as string;
        if (!System.Text.RegularExpressions.Regex.IsMatch(phone, @"^\d{3}-\d{3}-\d{3}$"))
            return new ValidationResult(false, Description);

        return ValidationResult.ValidResult;
    }
}

6. Podłączenie danych i logiki w klasie okna

Bindowanie ViewModelu i wywołanie asynchroniczne

Zadanie wymaga, by DataContext głównego okna był ustawiony na instancję klasy BookViewModel. Przycisk (z błyskawicą) wywołuje metodę LoadBooksAsync(), która zaczyta dane.

Plik: MainWindow.xaml.cs
using System.Windows;

namespace MiNIPotter
{
    public partial class MainWindow : Window
    {
        public MainWindow()
        {
            InitializeComponent();
            // Ustawienie ViewModelu wymaganego w poleceniu zadania
            DataContext = new BookViewModel();
        }

        // Zdarzenie Click dla przycisku z błyskawicą 🗲
        private async void LoadBtn_Click(object sender, RoutedEventArgs e)
        {
            if (DataContext is BookViewModel vm)
            {
                await vm.LoadBooksAsync();
            }
        }
    }
}

7. Konwertery (Ocena na gwiazdki i kolor RGB)

Jak przetłumaczyć liczbę 7.5 na ocenę w gwiazdkach i obliczyć kolor RGB?

W WPF nie możemy wpisać matematyki do XAML-a. Pisząc kod, używamy interfejsu IValueConverter, który pośredniczy pomiędzy daną wejściową (np. ocena w Double) a daną wyjściową dla UI (np. ciąg gwiazdek "★★★☆☆" lub kolor pędzla SolidColorBrush).

  • RatingToStarsConverter: Tworzy napis ze znakiem "★" oraz z dopełnieniem "☆" na podstawie formuły Math.Clamp.
  • AverageRatingToStarsConverter: Oblicza średnią ze zbioru ICollection<Chapter> dla konkretnej książki.
  • RatingToColorConverter: Oblicza dynamicznie kanały Red i Green zgodnie z zadaniem (wartości od 0 do 192).
Plik: Converters.cs (Stwórz taki plik w projekcie)
using System;
using System.Collections.Generic;
using System.Globalization;
using System.Linq;
using System.Windows.Data;
using System.Windows.Media;
using Potter.API;

namespace MiNIPotter
{
    public class RatingToStarsConverter : IValueConverter
    {
        public object Convert(object value, Type targetType, object parameter, CultureInfo culture)
        {
            if (value is double rating)
            {
                int fullStars = (int)Math.Clamp(rating, 0, 10);
                return new string('★', fullStars) + new string('☆', 10 - fullStars);
            }
            return "";
        }

        public object ConvertBack(object value, Type targetType, object parameter, CultureInfo culture) => throw new NotImplementedException();
    }

    public class AverageRatingToStarsConverter : IValueConverter
    {
        public object Convert(object value, Type targetType, object parameter, CultureInfo culture)
        {
            if (value is ICollection<Chapter> chapters && chapters.Any())
            {
                double avg = chapters.Average(c => c.Rating);
                int fullStars = (int)Math.Clamp(avg, 0, 10);
                return new string('★', fullStars) + new string('☆', 10 - fullStars);
            }
            return new string('☆', 10);
        }

        public object ConvertBack(object value, Type targetType, object parameter, CultureInfo culture) => throw new NotImplementedException();
    }

    public class RatingToColorConverter : IValueConverter
    {
        public object Convert(object value, Type targetType, object parameter, CultureInfo culture)
        {
            if (value is double rating)
            {
                byte r = (byte)((1 - (rating / 10)) * 192);
                byte g = (byte)((rating / 10) * 192);
                byte b = 0;
                return new SolidColorBrush(Color.FromRgb(r, g, b));
            }
            return Brushes.Black;
        }

        public object ConvertBack(object value, Type targetType, object parameter, CultureInfo culture) => throw new NotImplementedException();
    }
}

8. Główny Layout XAML (Wszystko w jednym)

Kluczowe rozwiązania w kodzie:

  • Import Słownika: <ResourceDictionary Source="Styles/Colors/Colors.xaml"/> wstrzykuje kolory, pozwalając na użycie {DynamicResource PrimaryBackgroundBrush}.
  • Zewnętrzna Czcionka: Zdefiniowana jako <FontFamily x:Key="HarryPotterFont">./#Harry Potter</FontFamily>. Odnosi się to bezpośrednio do pliku TTF w głównym katalogu projektu.
  • Formatowanie Daty: StringFormat='{}{0:d MMMM yyyy}' wymusza specyficzny, pełny format daty wymagany w punkcie dla kafelków z okładkami.
  • Expander (Rozdziały): Zamknęliśmy element wewnątrz `Expander`, gdzie nagłówek "Header" przejmuje tytuł rozdziału, a jego rozwinięcie ujawnia szczegółowe opisy i dynamicznie pokolorowane gwiazdki.
Plik: MainWindow.xaml
<Window x:Class="MiNIPotter.MainWindow"
        xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        xmlns:local="clr-namespace:MiNIPotter"
        Title="MiNIPotter Library"
        Width="1200" Height="800"
        MinWidth="1000" MinHeight="600"
        Background="{DynamicResource PrimaryBackgroundBrush}"
        Foreground="{DynamicResource PrimaryForegroundBrush}"
        Icon="potter.png">

    <Window.Resources>
        <ResourceDictionary>
            <!-- Wciąganie dostarczonych pędzli i kolorów -->
            <ResourceDictionary.MergedDictionaries>
                <ResourceDictionary Source="Styles/Colors/Colors.xaml"/>
            </ResourceDictionary.MergedDictionaries>

            <!-- Podłączanie lokalnych klas jako narzędzia konwertujące -->
            <local:RatingToStarsConverter x:Key="RatingToStarsConverter"/>
            <local:AverageRatingToStarsConverter x:Key="AverageRatingToStarsConverter"/>
            <local:RatingToColorConverter x:Key="RatingToColorConverter"/>

            <!-- Deklaracja czcionki pobranej z pliku .ttf -->
            <FontFamily x:Key="HarryPotterFont">./#Harry Potter</FontFamily>
        </ResourceDictionary>
    </Window.Resources>

    <Grid>
        <Grid.RowDefinitions>
            <RowDefinition Height="Auto"/> <!-- Sekcja z banerem i przyciskiem -->
            <RowDefinition Height="*"/>    <!-- Reszta okna -->
        </Grid.RowDefinitions>

        <!-- =================== TOP BANNER =================== -->
        <Grid Grid.Row="0" Margin="10">
            <TextBlock Text="MiNIPotter Library"
                       FontFamily="{StaticResource HarryPotterFont}"
                       FontSize="40"
                       FontWeight="Bold"
                       HorizontalAlignment="Center"
                       Foreground="{DynamicResource PrimaryForegroundBrush}"/>

            <Button HorizontalAlignment="Right" VerticalAlignment="Center"
                    Width="40" Height="40"
                    Background="{DynamicResource SecondaryBackgroundBrush}"
                    Foreground="{DynamicResource SecondaryForegroundBrush}"
                    Content="🗲" FontSize="16" FontWeight="Bold"
                    Click="LoadBtn_Click"/>
        </Grid>

        <!-- =================== GŁÓWNA ZAWARTOŚĆ =================== -->
        <Grid Grid.Row="1">
            <Grid.ColumnDefinitions>
                <ColumnDefinition Width="400"/> <!-- Sztywna szerokość dla Listy -->
                <ColumnDefinition Width="*"/>   <!-- Reszta na Szczegóły -->
            </Grid.ColumnDefinitions>

            <!-- ==== LISTA KSIĄŻEK ==== -->
            <ListBox Grid.Column="0" ItemsSource="{Binding Books}" x:Name="BooksList"
                     Background="{DynamicResource SecondaryBackgroundBrush}"
                     ScrollViewer.HorizontalScrollBarVisibility="Disabled"
                     ScrollViewer.VerticalScrollBarVisibility="Visible">
                <ListBox.ItemTemplate>
                    <DataTemplate>
                        <Grid Margin="5">
                            <Grid.ColumnDefinitions>
                                <ColumnDefinition Width="100"/>
                                <ColumnDefinition Width="*"/>
                            </Grid.ColumnDefinitions>

                            <Image Source="{Binding Cover}" Stretch="Uniform" Width="100"/>

                            <Grid Grid.Column="1" Margin="10,0,0,0">
                                <Grid.RowDefinitions>
                                    <RowDefinition Height="Auto"/>
                                    <RowDefinition Height="Auto"/>
                                    <RowDefinition Height="*"/>
                                    <RowDefinition Height="Auto"/>
                                </Grid.RowDefinitions>

                                <TextBlock Text="{Binding Title}" TextWrapping="Wrap" FontWeight="Bold" FontSize="18"/>
                                <TextBlock Grid.Row="1" Text="{Binding Author}" FontSize="10"/>

                                <StackPanel Grid.Row="3" Orientation="Horizontal" VerticalAlignment="Bottom">
                                    <TextBlock Text="{Binding Pages, StringFormat='Strony: {0}'}" FontSize="12" Margin="0,0,10,0"/>
                                    <TextBlock Text="{Binding ReleaseDate, StringFormat='{}{0:d MMMM yyyy}'}" FontSize="12"/>
                                </StackPanel>
                            </Grid>
                        </Grid>
                    </DataTemplate>
                </ListBox.ItemTemplate>
            </ListBox>

            <!-- ==== SZCZEGÓŁY WYBRANEJ KSIĄŻKI ==== -->
            <!-- Kontekst bindowania zostaje uwiązany ze wskazanym elementem na BooksList -->
            <Grid Grid.Column="1" DataContext="{Binding SelectedItem, ElementName=BooksList}" Margin="10,0,10,10">
                <Grid.RowDefinitions>
                    <RowDefinition Height="*"/> <!-- Detale -->
                    <RowDefinition Height="*"/> <!-- Rozdziały -->
                </Grid.RowDefinitions>

                <!-- Górna część: Detale + Okładka -->
                <Grid Grid.Row="0" Margin="0,0,0,10">
                    <Grid.ColumnDefinitions>
                        <ColumnDefinition Width="*"/>
                        <ColumnDefinition Width="*"/>
                    </Grid.ColumnDefinitions>

                    <ScrollViewer Grid.Column="0" VerticalScrollBarVisibility="Auto" Margin="0,0,10,0">
                        <StackPanel>
                            <TextBlock Text="{Binding Dedication}" FontSize="14" FontStyle="Italic" FontWeight="DemiBold" TextWrapping="Wrap"/>
                            <TextBlock Text="{Binding Author}" FontSize="14" HorizontalAlignment="Right" Margin="0,10,0,10"/>
                            <TextBlock Text="{Binding Summary}" TextWrapping="Wrap" FontSize="14" FontStyle="Italic"/>

                            <!-- Wywołanie naszego konwertera wyliczającego średnią i zwracającego ciąg gwiazdek -->
                            <TextBlock Text="{Binding Chapters, Converter={StaticResource AverageRatingToStarsConverter}}"
                                       FontSize="20" Foreground="Gold" Margin="0,10,0,0"/>
                        </StackPanel>
                    </ScrollViewer>

                    <Image Grid.Column="1" Source="{Binding Cover}" Stretch="Uniform"/>
                </Grid>

                <!-- Dolna część: Lista Rozdziałów -->
                <Grid Grid.Row="1" Background="{DynamicResource SecondaryBackgroundBrush}">
                    <Grid.RowDefinitions>
                        <RowDefinition Height="Auto"/>
                        <RowDefinition Height="*"/>
                    </Grid.RowDefinitions>

                    <TextBlock Text="Rozdziały" Foreground="{DynamicResource PrimaryForegroundBrush}" FontSize="20" FontWeight="Bold" Margin="10"/>

                    <ScrollViewer Grid.Row="1" VerticalScrollBarVisibility="Auto">
                        <ItemsControl ItemsSource="{Binding Chapters}">
                            <ItemsControl.ItemTemplate>
                                <DataTemplate>
                                    <!-- Element Expander - zwinięty / rozwinięty kafelek -->
                                    <Expander Header="{Binding Title}" FontSize="14" FontWeight="DemiBold" IsExpanded="False" Margin="5">
                                        <StackPanel Margin="10,5,10,10">
                                            <TextBlock Text="{Binding Summary}" TextWrapping="Wrap" FontSize="12" FontStyle="Italic"/>

                                            <!-- Gwiazdki + Konwerter koloru na bazie oceny od 0 do 10 -->
                                            <TextBlock Text="{Binding Rating, Converter={StaticResource RatingToStarsConverter}}"
                                                       Foreground="{Binding Rating, Converter={StaticResource RatingToColorConverter}}"
                                                       FontSize="16" Margin="0,5,0,0"/>
                                        </StackPanel>
                                    </Expander>
                                </DataTemplate>
                            </ItemsControl.ItemTemplate>
                        </ItemsControl>
                    </ScrollViewer>
                </Grid>
            </Grid>
        </Grid>
    </Grid>
</Window>
