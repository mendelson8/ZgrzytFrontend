Refleksja (Reflection) - Co trzeba umieć

Wymagane przestrzenie nazw:

using System;
using System.Reflection;
using System.Linq;

1. Tworzenie własnych atrybutów

Atrybut to klasa dziedzicząca po System.Attribute. Używamy dekoratora AttributeUsage, aby określić, gdzie atrybut może być stosowany (np. na klasach, właściwościach).

[AttributeUsage(AttributeTargets.Class | AttributeTargets.Property)]
public class MojAtrybut : Attribute
{
    public string Opis { get; }

    public MojAtrybut(string opis)
    {
        Opis = opis;
    }
}

2. Pobieranie z Assembly typów oznaczonych atrybutem

Należy pobrać bieżące Assembly, pobrać wszystkie typy, a następnie przefiltrować je używając GetCustomAttribute.

var assembly = Assembly.GetExecutingAssembly();

var znalezioneTypy = assembly.GetTypes()
    .Where(t => t.GetCustomAttribute<MojAtrybut>() != null);

3. Pobieranie typów dziedziczących po klasie abstrakcyjnej

Używamy metody IsSubclassOf. Warto pamiętać, aby odfiltrować samą klasę abstrakcyjną (!IsAbstract).

var typBazowy = typeof(BazaRaportu);

var raporty = assembly.GetTypes()
    .Where(t => t.IsSubclassOf(typBazowy) && !t.IsAbstract);

4. Pobieranie z typu właściwości oznaczonych atrybutem

Używamy metody GetProperties() na konkretnym typie i filtrujemy wynik.

var typ = typeof(Uzytkownik);

var props = typ.GetProperties()
    .Where(p => p.GetCustomAttribute<MojAtrybut>() != null);

5. Czy typ implementuje generyczny interfejs?

Sprawdzenie czy klasa implementuje interfejs typu IRepository<> (open generic) wymaga sprawdzenia definicji typu generycznego.

bool CzyToGeneryk(Type typ, Type szukanyInterfejs)
{
    // szukanyInterfejs to np. typeof(IRepository<>)
    return typ.GetInterfaces().Any(i =>
        i.IsGenericType &&
        i.GetGenericTypeDefinition() == szukanyInterfejs);
}

6. Ustawienie wartości właściwości przez .SetValue()

Wymaga instancji obiektu oraz obiektu PropertyInfo. Nazwę właściwości podajemy jako string.

var produkt = new Produkt(); // Instancja
var propInfo = typeof(Produkt).GetProperty("Cena");

if (propInfo != null)
{
    // SetValue(obiekt, nowaWartosc)
    propInfo.SetValue(produkt, 99.99);
}
