2. Typografia i Tekst

Pkt 3, 4, 5
Wymaganie 5 (Lucida Sans -> Verdana -> sans-serif) realizujemy przez listę fallbacków. Wymaganie 4 (Justowanie) i 3 (Środkowanie nagłówków) to proste właściwości text-align. 
/* style.css */
body {
    /* Pkt 5: Lucida Sans, jeśli brak to Verdana, jeśli brak to domyślny */
    font-family: "Lucida Sans", Verdana, sans-serif;

    /* Pkt 3: Nagłówki wyśrodkowane w kontenerach */
    text-align: center;
}

p {
    /* Pkt 4: Treść artykułów wyjustowana */
    text-align: justify;
}

4. Obrazek i Oblewanie Tekstem

Pkt 8
Aby tekst otaczał obrazek, używamy float: left lub right. Do adnotacji (podpisu) najlepszy jest tag <figure>. 
<!-- HTML -->
<figure class="lead-image">
    <img src="couple.jpg" alt="Description">
    <figcaption>Adnotacja pod obrazkiem</figcaption>
</figure>

/* CSS */
.lead-image {
    float: left; /* Pkt 8: Tekst otacza obrazek */
    margin: 0 15px 10px 0;
    width: 200px;
}
.lead-image img { width: 100%; }

5. Paywall: Czerwony Box

Pkt 9, 10, 11
Box musi być wyśrodkowany nad artykułem. Używamy position: absolute względem rodzica (relative). Przezroczystość 20% tła uzyskujemy przez rgba(..., 0.8) (0.8 to krycie, czyli 0.2 to przezroczystość). 
/* CSS */
.article-container {
    position: relative; /* Rodzic musi być relative */
}

.paywall-info {
    /* Pkt 9: Wyśrodkowany box */
    position: absolute;
    top: 50%; left: 50%;
    transform: translate(-50%, -50%);
    z-index: 10;

    /* Pkt 10: Czerwony, lekko zaokrąglone rogi */
    background-color: rgba(255, 0, 0, 0.8); /* Pkt 11: 20% przezroczystości */
    border-radius: 10px;
    padding: 20px;

    /* Pkt 11: Biały kolor tekstu */
    color: white;
    text-align: center;
    font-weight: bold;
}

6. Progresywne Rozmycie (Blur)

Pkt 12
To najtrudniejszy punkt. Używamy backdrop-filter na nakładce, która ma maskę gradientową. Dzięki temu rozmycie na dole jest silniejsze niż na górze. 
/* CSS */
.blur-layer {
    position: absolute;
    bottom: 0; left: 0;
    width: 100%; height: 60%;

    /* Pkt 12: Rozmycie */
    backdrop-filter: blur(10px);
    -webkit-backdrop-filter: blur(10px);

    /* Nasilanie się ku dołowi (Maska gradientowa) */
    mask-image: linear-gradient(to bottom, transparent, black);
    -webkit-mask-image: linear-gradient(to bottom, transparent, black);
}

7. Święty Graal Layoutu: Flexbox vs Grid

Złota zasada: Flexbox służy do układania elementów w jednym wymiarze (tylko w rzędzie LUB tylko w kolumnie). Grid służy do budowania w dwóch wymiarach naraz (rzędy I kolumny, jak szachownica).

A. Flexbox (display: flex)

Flexbox rozwiązuje odwieczny problem centrowania w pionie i poziomie w CSS. Kiedy nadajesz kontenerowi display: flex, jego dzieci stają się elastyczne. Działasz na dwóch osiach: osi głównej (domyślnie poziomej - justify-content) i osi poprzecznej (domyślnie pionowej - align-items).

Przykład: Idealne wyśrodkowanie i rozkład elementów

<!-- HTML -->
<div class="flex-container">
    <div class="box">A</div>
    <div class="box">B</div>
    <div class="box">C</div>
</div>

/* CSS */
.flex-container {
    display: flex;

    /* flexDirection: row (domyślnie - elementy obok siebie) lub column (elementy pod sobą) */
    flex-direction: row;

    /* justify-content: Rozkład na osi GŁÓWNEJ (tutaj poziomej).
       Opcje: center, space-between (maksymalnie rozsuwa), space-around, flex-start, flex-end */
    justify-content: space-between;

    /* align-items: Rozkład na osi POPRZECZNEJ (tutaj pionowej).
       Opcje: center, flex-start (do góry), flex-end (do dołu), stretch (rozciągnięcie) */
    align-items: center;

    /* gap: nowsza właściwość, ustala równy odstęp między elementami bez marginesów! */
    gap: 20px;

    /* flex-wrap: wrap pozwala elementom spaść do nowej linii, gdy brakuje miejsca */
    flex-wrap: wrap;
}

/* Magia flex na dzieciach */
.box {
    /* flex: 1 sprawia, że element zajmie całą DOSTĘPNĄ wolną przestrzeń.
       Gdy 3 elementy mają flex: 1, podzielą się kontenerem po równo (1/3). */
    flex: 1;
}

B. CSS Grid (display: grid)

Grid to potężna siatka. Ty definiujesz strukturę kolumn i rzędów, a elementy po prostu "wpadają" w wyznaczone kratki lub mogą rozciągać się na kilka kratek naraz. Idealne do budowania całych szablonów stron (Header, Sidebar, Main, Footer).

Przykład: Szachownica z boczkiem (Układ klasycznej strony)

<!-- HTML -->
<div class="grid-container">
    <header class="header">NAGŁÓWEK</header>
    <aside class="sidebar">MENU</aside>
    <main class="content">TREŚĆ ARTYKUŁU</main>
    <footer class="footer">STOPKA</footer>
</div>

/* CSS */
.grid-container {
    display: grid;

    /* Definiujemy kolumny. Tutaj tworzymy dwie:
       Pierwsza ma sztywno 250px.
       Druga zajmuje 1fr (1 fraction - całą resztę dostępnego miejsca) */
    grid-template-columns: 250px 1fr;

    /* Definiujemy rzędy. Tutaj trzy:
       auto (dopasuj do zawartości), 1fr (zajmij resztę miejsca), 50px (sztywne dla stopki) */
    grid-template-rows: auto 1fr 50px;

    gap: 15px; /* Odstęp między kratkami */
}

/* Wrzucanie elementów w konkretne miejsca w siatce */
.header {
    /* Mówimy: zacznij od 1 linii pionowej i skończ na 3 linii pionowej
       (czyli rozciągnij się na obie kolumny!) */
    grid-column: 1 / 3;
}

.sidebar {
    /* Nie musimy nic pisać! Wpadnie automatycznie w pierwszy wolny slot (lewa kolumna, drugi rząd) */
}

.content {
    /* Wpadnie automatycznie obok sidebara (prawa kolumna, drugi rząd) */
}

.footer {
    grid-column: 1 / 3; /* Znowu rozciągamy na całą szerokość */
}
